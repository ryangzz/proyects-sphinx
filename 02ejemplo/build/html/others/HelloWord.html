Variables¶

Una vez hemos guardado un valor en una variable, podemos hacer referencia a él a lo largo del programa. Por ejemplo, el siguiente programa … Ademas no es necesario el uso del aterrador punto y coma

a = 2
b = 3
c = a + b
print(c)

  • guarda el valor 3 en la variable b

  • guarda el valor 2 en la variable a
